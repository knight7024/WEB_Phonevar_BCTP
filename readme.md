## Phonevar란?
장병들의 주도적인 스마트폰 사용 및 관리를 자동으로 도와주는 서비스 앱
<br>

---
## 아이디어
스마트폰의 끄고 켜짐을 기준으로 사용기록을 기록해 사용자가 정상적인 사용을 하고 있는지 비정상적인 사용을 하고 있는지 관리자가 사용기록을 확인하여 사용위한 사용자를 식별하는 아이디어
<br>

---
## 프로젝트 기능 설계
프로젝트는 크게 3가지 구분으로 개발 되었고, 각 구분은 다음과 같은 기능이 있습니다.
* 사용자 앱 / 관리자 앱
  * 사용자 등록
  * 사용기록 작성
  * 스마트폰 제출 푸시 알림
  * 사용기록 서버 전송

* 서버
  * 사용기록 검증
  * 관리자 푸시알림

* 웹페이지
  * 사용기록 열람
  * 스마트폰 사용시간 수정
  * 사용자상태 변경(정상/비정상)
<br>
